# Tecnólogo Suporte - Landing Page

Esta é uma landing page profissional desenvolvida para o setor de assistência técnica de smartphones e computadores. O projeto foi criado focado em conversão e experiência do usuário (UX), garantindo que clientes encontrem soluções rápidas para seus dispositivos.

## Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias "puro sangue" (Vanilla), demonstrando domínio de estrutura e estilização sem dependência de frameworks:

- **HTML5:** Estruturação semântica para SEO.
- **CSS3:** Estilização moderna utilizando Flexbox e Grid para responsividade.
- **JavaScript:** Manipulação de DOM para interatividade (como botões e formulários).

## Funcionalidades

- **Design Responsivo:** O site se adapta perfeitamente a qualquer tamanho de tela (Celulares, Tablets e Desktops).
- **Seção de Serviços:** Listagem clara de reparos para celulares e PCs.
- **Call to Action (CTA):** Botões estratégicos para contato via WhatsApp.
- **Otimização de Performance:** Código limpo para carregamento rápido.
